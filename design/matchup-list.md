# マッチアップリスト デザイン仕様 (#73)

## 概要

Viewers ページ上部に表示される5行のマッチアップリスト。味方チャンピオンと敵チャンピオンの対面ペアを一覧表示し、ビューアの操作を支援する。

## レイアウト仕様

Issue #73 のデザインモックアップに基づく寸法:

| 要素 | 高さ |
|------|------|
| セクションタイトル | 33px |
| 各行 (アイテム) | 49px |
| リスト全体 | 244px |

**備考**: デザインモックアップにはウィンドウオープンボタンが含まれていないが、実装では必要。

## 現在の実装

### 配置

```
Viewers ページ
├─ MatchupListWidget (matchup_list_widget)  ← ここ
├─ Toolbar
└─ Viewers Splitter
```

### 行の構成 (左から右)

```
[▲] [▼] [味方アイコン] [味方チャンピオン名] [⇄] [敵チャンピオン名] [敵アイコン] [↗]
```

| 要素 | サイズ | 説明 |
|------|--------|------|
| ▲ / ▼ ボタン | 20×20px | 行の並べ替え (#67) |
| チャンピオンアイコン | 24×24px | チャンピオン画像 (Data Dragon) |
| チャンピオン名 (味方) | 可変幅 | 青色 (`#0078f5`) 9pt |
| ⇄ ボタン | 20×20px | 敵チャンピオンの入れ替え (#67) |
| チャンピオン名 (敵) | 可変幅 | 赤色 (`#e0342c`) 9pt |
| ↗ ボタン | 20×20px | マッチアップビューアを開く (#68) |

### スタイル

- **背景色**: `#090e14` (sidebar-background)
- **行の高さ**: 50px (`setFixedHeight`)
- **行の余白**: 左右 4px、上下 1px
- **行間**: 2px
- **コンテナ余白**: 左右 8px、上下 4px
- **ボタンスタイル**:
  - 背景: `#1c2330` (secondary)
  - テキスト: `#c1c9d4` (secondary-foreground)
  - ホバー: `#222a35` (border)
  - 角丸: 3px

### Feature Flag

- キー: `matchup_list`
- デフォルト: OFF (実験的機能)
- Settings > Feature Flags で切り替え可能

## 機能

### 行の並べ替え (#67)
- ▲ / ▼ ボタンで行を上下に移動
- 味方・敵ペアがそのまま移動する

### 敵チャンピオンの入れ替え (#67)
- ⇄ ボタンで現在行と次の行の敵チャンピオンを交換
- 味方チャンピオンは固定のまま

### ビューアを開く (#68)
- ↗ ボタンでその行のマッチアップをビューアで開く
- 行の位置からレーンを判定 (0=top, 1=jungle, 2=middle, 3=bottom, 4=support)

### データ連携

`ChampionDetectorService` の `matchup_pairs_updated` シグナルからデータを受信:

```
ChampionDetectorService.matchup_pairs_updated(pairs)
  → MainWindow.on_matchup_pairs_updated(pairs)
    → self._matchup_data を更新
    → update_matchup_list() でUI反映
```

## デザインモックアップとの差分

| 項目 | デザイン | 実装 |
|------|----------|------|
| 行の高さ | 49px | 50px |
| ウィンドウオープンボタン | なし | ↗ ボタンとして実装済み (#68) |
| 並べ替えボタン | なし | ▲/▼ ボタンとして実装済み (#67) |
| 敵入れ替えボタン | なし | ⇄ ボタンとして実装済み (#67) |
