# LoL Analytics Viewer

League of LegendsのチャンピオンビルドをLoLAnalyticsで確認するためのElectronアプリ。

## 機能

- チャンピオン名を入力するだけでLoLAnalyticsのビルドページを開く
- シンプルで使いやすいUI

## セットアップ

```bash
npm install
```

## 起動方法

```bash
npm start
```

## 使い方

1. アプリを起動
2. チャンピオン名を入力（例: Ashe, Yasuo, Jinx）
3. 「ビルドページを開く」ボタンをクリック、またはEnterキーを押す
4. デフォルトブラウザでLoLAnalyticsのページが開きます

## URL例

- Ashe: https://lolalytics.com/lol/ashe/build/
- Yasuo: https://lolalytics.com/lol/yasuo/build/
- Jinx: https://lolalytics.com/lol/jinx/build/
