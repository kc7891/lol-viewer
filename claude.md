# AI開発プロジェクトについて
このプロジェクトは **AIによる生成のみで成り立っている** プロジェクトです。
すべてのコード、ドキュメント、テスト、設定ファイルはAIアシスタントによって生成・保守されます。

# AI開発の原則

## 自律性と責任
- AIが自分でできることは **なるべく自分で実行する**
  - 実装の確認
  - テストの実行と結果確認
  - 動作確認
  - GitHub Actionsの確認
  - GitHub Actionsログの確認
  - キャッシュの確認
  - その他、検証可能なすべての項目

## 品質重視のアプローチ
- **実際に動かして確認するよりも、コードやドキュメントを注意深く構築・確認する** ことを優先
- 最低限の動作確認で正しい挙動をするアプリケーションを構築する
- 事前の設計と実装の正確性により、デバッグサイクルを最小化する

# 開発の前提
- Windowsユーザを想定している
- モダンでシンプルな構成で開発を行う
- 最小限で最大限効果がある実装を行う
- できる限りテストコードを実装する

# コミット時の注意
- テストを実行して通ってからコミットする
- 変更内容を十分に確認してからpushする
